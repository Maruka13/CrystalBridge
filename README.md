> **Projeto de lógica avançada**, focado no uso de loops e validações rigorosas para criar um jogo de sorte e estratégia baseado em níveis.

---

## 🚀 Como o jogo funciona

Inspirado em desafios de sobrevivência da cultura pop, a **Ponte de Cristal** testa a sorte e a atenção do jogador através de 3 rodadas consecutivas:

- **Níveis Progressivos:** O jogador deve atravessar 3 plataformas para vencer.
- **Lógica de Morte Súbita:** Um erro em qualquer nível encerra o jogo imediatamente através do comando `return`.
- **Validação de Entrada:** O sistema não aceita números fora do intervalo 1-3, forçando o jogador a escolher uma opção válida antes de prosseguir.

## 🧠 Conceitos de Programação Aplicados

Este projeto foi fundamental para consolidar o entendimento em:
1. **Laços de Repetição (`while`):** Utilizados tanto para gerenciar as rodadas quanto para criar um "loop de validação" que impede entradas incorretas.
2. **Geração Aleatória Dinâmica:** Uso de `Math.random()` e `Math.floor()` para definir qual piso está quebrado em cada nova tentativa.
3. **Escopo de Variáveis:** Gerenciamento da variável `rodada` para determinar a condição de vitória final.

## 🛠️ Tecnologias

- **HTML5/CSS3:** Interface temática para o desafio.
- **JavaScript (ES6):** Motor de jogo e lógica de repetição.

---

## ⚙️ Como Executar

1. Clone este repositório.
2. Abra o arquivo `index.html` no seu navegador preferido.
3. Clique em **"Bora jogar!"** e tente chegar ao outro lado da ponte sem quebrar o vidro!
